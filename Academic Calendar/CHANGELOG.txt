CHANGELOG
=========

v0.6.00 (Unreleased)
--------------------
- Reinstated markbook assessment rows that are linked to Planner entries.
- Added `Assessment Display Basis` setting for assessment labels and grouped summaries.
- Added `Merge Same-Day Assessments` setting for assessment event compaction.
- Added `Learning Area` support for merged assessment display.
- Improved merged assessment tooltips to show every merged assessment detail block.
- Added classification colour settings for formative, summative, and not-classified assessments.
- Added `Use Assessment Classification Colour in Calendar` toggle for assessment event colouring.

v0.5.00 (2026-04-22)
--------------------
- Fixed the dashboard hook configuration so Homework tabs display reliably again.
- Added Backup and Restore Settings to export and restore Academic Calendar `gibbonSetting` rows.
- Added Student and Parent permission availability for Summative Assessment Overview.
- Restricted Student and Parent overview access to only their allowed year groups.
- Added `Overview Week Number Mode` setting for the Summative Assessment Overview.
- Added support for showing either calendar weeks or academic weeks in the overview.
- Academic week numbering now starts at 1 and does not increment during full-school-closure weeks.
- Moved `Overview Week Number Mode` to the top of the Summative Assessment Overview settings block.
- Changed the default overview week-number mode to `Academic Week`.
- Improved settings page layout for the Summative Assessment Overview section.

v0.4.00 (2026-04-21)
--------------------
- Added scaffolding for a new `Summative Overview` page (weekly matrix layout).
- Added weekly summative assessment counts per enabled year group in `Summative Overview`.
- Added threshold-based highlighting for summative weekly counts (normal / at threshold / exceed).
- Added Manage Settings controls for default and per-year-group summative weekly thresholds.
- Added Summative Overview filters (term, year-group subset, and only-weeks-with-activity).
- Added new module action: `Summative Overview` (staff/admin-facing).
- Added settings for weekly summative threshold:
  - `summativeWeeklyThresholdDefault` (fallback, default `3`)
  - `summativeWeeklyThresholdByYearGroup` (per-year-group JSON map)
- Renamed display settings labels to be clearer:
  - Show Weekends on Calendar
  - Show Homework Events on Calendar
  - Show Assessment Events on Calendar

v0.3.00 (2026-04-20)
--------------------
- Added Markbook assessment events to the Homework Calendar feed (student, parent, staff).
- Assessment events are rendered as all-day entries in FullCalendar.
- Added module settings toggles:
  - Show Homework Events
  - Show Assessment Events
- Added role-aware assessment click-through URLs to Markbook pages where accessible.
- Updated staff year-group label rendering to prefer short names.
- Added visual differentiation between homework and assessment events.
- Added per-user formative/summative assessment filter controls on the calendar view.
- Added per-user `Not Classified` assessment filter control on the calendar view.
- Added event-type metadata settings (visibility + assessment classification) in Manage Settings.
- Cleaned up calendar event CSS to stabilize day-grid layout and prevent right-column overflow.

v0.2.0 (2026-04-19)
--------------------
- Added Markbook assessment events to the Homework Calendar feed (student, parent, staff).
- Assessment events are rendered as all-day entries in FullCalendar.
- Added module settings toggles:
  - Show Homework Events
  - Show Assessment Events
- Added role-aware assessment click-through URLs to Markbook pages where accessible.

v0.1.0 (2026-04-12)
-------------------
- Initial release of the Academic Calendar module.
- Added 'Homework Calendar' action for Staff, Students and Parents.
- Implemented FullCalendar month/week/list views with mobile-friendly toolbar behavior.
- Added calendar event source endpoint: 'calendar_eventsJSON.php'.
- Refactored data access into gateway class: 'src/Domain/AcademicCalendarEventGateway.php'.
- Added settings management for:
  - Event type colours (palette picker + custom hex input)
  - Show weekends
  - Default staff view
  - Enabled year groups
- Added dashboard hooks for Staff, Student and Parental dashboards.
- Added year-group filtering controls and permission-based all-year-group staff option.
